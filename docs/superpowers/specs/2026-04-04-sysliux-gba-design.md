# sysLiux-GBA 模拟器设计文档

## 1. 项目概述

**项目名称：** sysLiux-GBA
**类型：** Android GBA 模拟器
**核心功能：** 基于 mGBA 核心的 Game Boy Advance 模拟器，提供沉浸式游戏体验
**目标用户：** Android 手游玩家、掌机怀旧用户
**许可证说明：** mGBA 采用 MPL-2.0 许可证（本项目集成 mGBA 源码需遵守 MPL-2.0）

## 2. 技术架构

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 模拟器核心 | mGBA | 开源成熟，ARM7TDMI 模拟精准 |
| UI 层 | Kotlin + Jetpack Compose | 现代声明式 UI，Material Design 3 |
| 渲染 | OpenGL ES 3.0+ | 2D 纹理渲染，Framebuffer blit 加速 |
| 音频 | AAudio | 48kHz 采样，缓冲区 < 512 samples |
| 音视频同步 | Audio Track 主时钟 | 以音频回调为基准同步视频帧 |
| 存储 | Room + DataStore | 存档元数据 + 用户设置 |
| NDK | r25c+ | JNI 绑定层编译 |
| 最小 SDK | Android 8.0 (API 26) | 覆盖率 >99% 设备 |

### 2.1 JNI 集成策略

- **帧缓冲交换**：使用 `DirectBuffer` 避免数据拷贝，减少延迟
- **输入事件传递**：通过 JNI 回调将触控/手柄事件传入模拟器核心
- **子模块管理**：`mGBA` 作为 git submodule 存于 `jni/mgba/`

## 3. UI 设计

### 3.1 整体布局（Delta 霓虹复古风格）

```
┌─────────────────────────────────────┐
│  sysLiux-GBA              [≡] [▶]  │  ← 顶部栏
├─────────────────────────────────────┤
│                                     │
│                                     │
│          GBA 游戏画面                │  ← 游戏区域（整数倍缩放）
│         （240×160 原始比例）         │
│                                     │
│                                     │
├─────────────────────────────────────┤
│  [L] [R]         [A] [B] [X] [Y]   │  ← 底部遮罩栏
│            [SELECT][START]          │
│              [↑↓←→]                 │  ← 十字方向键
└─────────────────────────────────────┘
```

- **顶部栏**：游戏名称 + 菜单按钮 + 运行控制按钮（暂停/继续）
- **游戏区域**：
  - 竖屏（默认）：游戏画面 2x-4x 整数缩放居中（根据屏幕尺寸动态计算最大缩放）
  - 横屏：游戏画面旋转 90° 填充，UI 整体适配
  - 支持旋转锁定选项
- **底部遮罩栏**：半透明黑色背景（70%），固定虚拟按键

### 3.2 霓虹复古配色

| 元素 | 色值 | 用途 |
|------|------|------|
| 主背景 | #0D0D1A | 深紫黑底色 |
| 霓虹青 | #00FFFF | 强调色、选中态 |
| 霓虹粉 | #FF00FF | 按钮按下/选中时发光 |
| 电紫色 | #9D00FF | 装饰线条 |
| 淡紫白 | #E0E0FF | 主文字 |
| 遮罩栏 | #000000 (70%) | 底部按键背景 |

### 3.3 虚拟按键布局

| 按键 | 位置 |
|------|------|
| L / R | 遮罩栏左上/右上 |
| A / B / X / Y | 遮罩栏右上方 |
| SELECT / START | 遮罩栏中央 |
| 十字方向键 | 遮罩栏左下方 |

按键采用半透明玻璃态 + 霓虹边框发光效果（仅按下时发光，用 BlendMode.Add 实现）。

### 3.4 按键透明度

- **遮罩栏背景**：固定 70% 黑色
- **按键本身**：10%-80% 可调节（默认 50%）

```
│   │   │   └── navigation/      # 导航路由
│   │   ├── core/                # mGBA JNI 绑定层
│   │   ├── data/

│   │   │   └── preferences/     # DataStore 用户设置
│   │   └── di/                  # Hilt 依赖注入

```

### 5.1 模块依赖关系

```

     │                   ↓

     ↓
di ──────────→ 所有模块（依赖注入）
```

## 7. 关键实现决策

1. **mGBA 核心**：通过 JNI 集成原生 C 代码，NDK r25c+ 编译
2. **渲染**：OpenGL ES 3.0+ 进行 2D 纹理渲染，Framebuffer blit 硬件加速
3. **音频**：AAudio API，48kHz 采样，缓冲区 < 512 samples
4. **音视频同步**：Audio Track 作为主时钟（gallium reference clock），以音频回调为基准同步视频帧
5. **帧缓冲交换**：使用 `DirectBuffer` 避免数据拷贝
6. **输入延迟预算**：触控采样 8ms + 渲染 8ms = 目标 < 16ms
7. **状态管理**：Compose State + ViewModel，UI 状态与模拟器状态分离
8. **依赖注入**：Hilt，模块化管理 Core、Data、UI 依赖
